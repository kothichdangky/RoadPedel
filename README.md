# Mini CRUD RoadPedel Laravel Demo Project

Đây là project demo được xây dựng nhằm mục đích học tập và thực hành:
- Backend: Laravel
- Frontend: Laravel
- Authentication: Laravel Sanctum
- Database: MySQL
- Mail: Mailpit (local)
- Firebase, Paypal: chỉ dùng cho demo (không kèm key, nằm trong env)

Project **không phải hệ thống production**, dùng để thể hiện kỹ năng backend – frontend cơ bản.
